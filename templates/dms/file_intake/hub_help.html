Ayuda — Carga de archivos

Subida de muestras y archivos de producción para probar y ejecutar transformaciones.

← Volver a Archivo

Propósito

Recibe archivos desde el equipo local, valida tipo/tamaño y los guarda internamente. Las extensiones permitidas dependen del tipo de origen del proyecto ({{ accept_attr }}).

Dos usos

TipoPara qué sirveNotas
Muestra Probar parseo / previsualizar sin job de producción. Límite de tamaño menor; se puede previsualizar y eliminar.
Producción Crea un job listo para Ejecutar (transformación). Requiere versión publicada.

Indicadores

IndicadorSignificado
MuestrasArchivos de prueba guardados.
Jobs subidosArchivos de producción pendientes/listos de ejecución.
Versión publicadaSi no hay, no se puede subir producción.

Flujo típico

  1. Subir una muestra para validar el perfil de origen.
  2. Publicar la versión del proyecto.
  3. Subir producción y continuar en Ejecutar.

Ayuda — Editor de reglas

Pipeline ordenado por campo destino. Vacío = se escribe el valor del mapeo sin transformar.

← Volver al editor

Propósito

Para cada campo destino mapeado define una secuencia de operaciones (transform_pipeline). El orden importa: se aplican de arriba abajo.

Operaciones típicas (MVP)

OperaciónEfecto
trimQuita espacios al inicio/fin.
upper / lowerMayúsculas o minúsculas.
date_formatReformatea fechas según patrón.
pad_left / pad_rightRellena hasta una longitud.
default_if_emptyValor por defecto si queda vacío.

Acciones

AcciónEfecto
GuardarPersiste los pipelines en el borrador.
PreviewSi está disponible, prueba el efecto sobre un valor de ejemplo.
← Hub reglasVuelve al resumen del módulo.

Al terminar, puede publicar la versión en el hub del proyecto (junto con origen, destino y mapeo).
